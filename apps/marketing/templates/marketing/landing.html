Teknologi

Dibangun di Atas Teknologi Modern

Django 6 dan Python 3.13 — dilengkapi WebSocket real-time, task queue asinkron, dan monitoring produksi.

Django 6.0 + Python 3.13

Stack yang sama dengan Instagram dan Pinterest untuk melayani jutaan pengguna.

Django 6.0.3 ASGI + Daphne Celery + Redis PostgreSQL
Python + Django 6 Pilihan Kami

ASGI + Daphne menangani ribuan koneksi WebSocket bersamaan. Python 3.13 menghadirkan peningkatan kecepatan eksekusi. Celery + Redis mengelola proses berat secara asinkron.

Node.js (Express/Fastify) I/O-bound

RPS sangat tinggi untuk I/O ringan. Namun untuk aplikasi ujian yang butuh ORM relasional dan logika bisnis kompleks, ekosistem Node kurang terstruktur dibanding Django.

PHP (Laravel / CI) Umum Dipakai

Mayoritas CBT Indonesia berbasis PHP. Model PHP-FPM membuat setiap request spawn proses baru — throughput jauh di bawah ASGI Python saat ujian serentak ratusan siswa.

FAQ

Pertanyaan yang Sering Ditanyakan

Apakah Rp499.000 sudah termasuk pemasangan?
Ya, harga Rp499.000 sudah termasuk jasa pemasangan dan konfigurasi awal di server Anda. Yang perlu Anda siapkan sendiri hanya VPS dan domain.
Berapa spesifikasi VPS minimal yang dibutuhkan?
Untuk penggunaan standar hingga 200 peserta serentak, VPS dengan 2 vCPU dan 2 GB RAM sudah cukup. Kami bantu rekomendasikan spesifikasi yang sesuai saat konsultasi.
Apakah ada biaya tambahan setelah pembelian?
Tidak ada biaya langganan bulanan atau tahunan. Update fitur dan perbaikan bug juga gratis. Biaya rutin hanya VPS dan domain yang Anda kelola sendiri.
Berapa lama proses pemasangan?
Biasanya selesai dalam 1 hari kerja setelah Anda menyediakan akses server dan domain. Kami memandu seluruh prosesnya melalui WhatsApp.
Apakah bisa diakses dari HP siswa?
Ya, CBT Pro sepenuhnya responsif. Siswa bisa mengerjakan ujian dari smartphone Android maupun iOS, tablet, atau laptop — tanpa perlu menginstal aplikasi apapun.
