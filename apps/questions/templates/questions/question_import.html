Panduan Format Data

Template Excel menggunakan header teknis pada baris pertama, misalnya subject, question_type, dan question_text.

Template Bank Soal

Field wajib:

  • subject, question_type, question_text
  • Untuk multiple_choice: isi minimal dua opsi dan correct_option.
  • Untuk checkbox: isi minimal dua opsi, correct_option atau correct_options dengan format A,C, dan opsional checkbox_scoring.
  • Untuk ordering: isi ordering_items dalam format JSON array, minimal dua item.
  • Untuk matching: isi matching_pairs dalam format JSON array, minimal dua pasangan.
  • Untuk fill_in_blank: gunakan placeholder {{1}}, {{2}}, dst di question_text, lalu isi blank_answers.
  • Untuk essay dan short_answer: isi answer_text.

Field tambahan yang didukung:

category, difficulty_level, points, explanation, question_image_url, audio_play_limit,
video_play_limit, allow_previous, allow_next, force_sequential, time_limit_seconds,
option_a, option_b, option_c,
option_d, option_e, option_f, option_g, option_h, option_i, option_j,
correct_option, correct_options, checkbox_scoring, ordering_items, matching_pairs,
blank_answers, answer_text, keywords, is_case_sensitive, max_word_count, tags, is_active

Aturan Nilai (Enum/Boolean)

  • difficulty_level: easy, medium, hard.
  • question_type: multiple_choice, checkbox, ordering, matching, fill_in_blank, essay, short_answer.
  • audio_play_limit, video_play_limit: angka bulat lebih dari 0, kosongkan jika tanpa batas.
  • allow_next, allow_previous, force_sequential, is_case_sensitive, is_active: isi TRUE atau FALSE.
  • correct_option: isi huruf kapital A sampai J untuk soal pilihan ganda.
  • ordering_items: contoh [{"text":"Langkah 1","order":1},{"text":"Langkah 2","order":2}].
  • matching_pairs: contoh [{"prompt":"Indonesia","answer":"Jakarta","order":1},{"prompt":"Jepang","answer":"Tokyo","order":2}].
  • blank_answers: contoh {"1":["Air"],"2":{"accepted_answers":["H2O"],"blank_points":2}}.
  • subject: gunakan nama atau kode subject aktif yang ada di sistem.
